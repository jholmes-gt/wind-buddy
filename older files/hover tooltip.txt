Creating a hover tooltip with JavaScript involves using event listeners to display and hide a custom tooltip element. This approach gives you full control over the styling and positioning, unlike the browser's default title attribute. 
Example: Simple tooltip using HTML, CSS, and JavaScript
1. HTML: Add a data attribute for the tooltip text 
Instead of the title attribute, use a custom data-* attribute (e.g., data-tooltip) to store the tooltip's content. This allows you to manage it with JavaScript and prevents the browser's default tooltip from appearing. 
html
<p>
  This is some text with an important
  <strong class="tooltip-trigger" data-tooltip="This is the text for my custom tooltip!">word</strong>
  in the middle.
</p>
Use code with caution.

2. CSS: Style the tooltip
The CSS hides the tooltip by default and styles its appearance. The tooltip is positioned absolutely relative to its parent container, the body. 
css
.tooltip {
  display: none; /* Hidden by default */
  position: absolute;
  padding: 8px 12px;
  background-color: #333;
  color: #fff;
  border-radius: 4px;
  font-size: 14px;
  white-space: nowrap; /* Prevents text from wrapping */
  z-index: 1000; /* Ensures it appears on top */
  pointer-events: none; /* Allows mouse to hover over content behind tooltip */
}
Use code with caution.

3. JavaScript: Handle the events and positioning 
The JavaScript handles the mouseover and mouseout events on the trigger element. It dynamically creates, positions, and removes the tooltip element. 
javascript
document.addEventListener('DOMContentLoaded', () => {
  let tooltip = null;

  function showTooltip(event) {
    // Get the tooltip text from the data attribute
    const tooltipText = event.target.getAttribute('data-tooltip');
    if (!tooltipText) return;

    // Create the tooltip element
    tooltip = document.createElement('div');
    tooltip.className = 'tooltip';
    tooltip.textContent = tooltipText;
    document.body.appendChild(tooltip);

    // Position the tooltip based on the mouse position
    positionTooltip(event);
  }

  function hideTooltip() {
    if (tooltip) {
      tooltip.remove();
      tooltip = null;
    }
  }

  function positionTooltip(event) {
    if (!tooltip) return;

    // Get the dimensions and position of the trigger element
    const triggerRect = event.target.getBoundingClientRect();

    // Position the tooltip near the trigger element
    const tooltipTop = triggerRect.top + window.scrollY - tooltip.offsetHeight - 10;
    const tooltipLeft = triggerRect.left + window.scrollX + (triggerRect.width / 2) - (tooltip.offsetWidth / 2);
    
    tooltip.style.top = `${tooltipTop}px`;
    tooltip.style.left = `${tooltipLeft}px`;
  }

  // Find all elements with the 'tooltip-trigger' class
  const triggers = document.querySelectorAll('.tooltip-trigger');

  // Add event listeners to each trigger
  triggers.forEach(trigger => {
    trigger.addEventListener('mouseover', showTooltip);
    trigger.addEventListener('mousemove', positionTooltip); // Continuously update position
    trigger.addEventListener('mouseout', hideTooltip);
  });
});
Use code with caution.

How the JavaScript works
The script waits for the DOMContentLoaded event to ensure all HTML elements are loaded before it runs.
It selects all elements with the class tooltip-trigger.
An event listener for mouseover is added to each trigger element:
It reads the tooltip content from the data-tooltip attribute.
It creates a new div with the class tooltip.
It appends this new div to the body of the document.
A mousemove listener continuously adjusts the tooltip's position relative to the cursor, ensuring it stays close to the trigger element as the mouse moves.
A mouseout listener is added to remove the tooltip div when the cursor leaves the trigger element, resetting the process for the next hover.
Helper functions showTooltip, hideTooltip, and positionTooltip are defined to keep the logic clean and reusable.
